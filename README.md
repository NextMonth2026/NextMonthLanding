# NextMonth Landing

Static Vite + React + Tailwind + Framer Motion landing page for NextMonth.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is written to `dist/`.

## Cloudflare Pages

- Framework preset: Vite
- Build command: `npm run build`
- Output dir: `dist`
- Node: 18+
